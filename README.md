# GIS Portfolio - Frontend Assignment

Project converted from a Figma design into semantic HTML and vanilla CSS. Responsive for desktop, tablet and mobile.

 ## How to run
 - Open index.html in a browser (no server required).

 ## Breakpoints used
 - Desktop: 1440px and up
 - Tablet: up to 1024px (target 768px)
 - Mobile: up to 480px (target 375px)

 ## What I implemented
 - Semantic HTML5 structure: `header`, `nav`, `main`, `section`, `footer`.
 - Responsive layout using CSS Grid and Flexbox.
 - CSS variables for colors and spacing in `css/style.css`.
 - Meaningful alt text for images and accessible headings.

 ## Challenges
 - Exact typography and spacing from the Figma file may require additional asset exports. I used the provided image and created placeholder project cards.
